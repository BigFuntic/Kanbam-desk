# Kanban Board App

A simple web-based task management application in kanban board format.
It supports the creation of whiteboards, columns and cards, drag & drop and synchronization between tabs.

Technologies used:
React,
Vite,
JavaScript, 
CSS (custom themes),
localStorage / sessionStorage,
BroadcastChannel API,
dnd-kit

## Installation and launch

❗To use the application, you need to have an installed Node.js on your device❗

Download Node.js is available via the link:

```https://nodejs.org/en/download```

1. Clone the repository

```https://github.com/BigFuntic/Kanbam-desk.git```

```cd Kanbam-desk```

2. Install dependencies

```npm install ```

3. Launch the project

```npm run dev ```

4. Open in the browser

```o```

By default:

```http://localhost:5173```

## How to use it

1. Enter the user's name (any text)
2. Create a board
3. Add columns
4. Add cards
5. Drag the cards between the columns
